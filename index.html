FAIRE UNE PAGE WEB ET L'HEBERGER EN DIX MINUTES C'EST POSSIBLE!

Pour effectuer ce tutoriel aucune connaissance à la programmation informatique n'est requise, nullement besoin d'installer un logiciel, avoir un pc et une connexion internet c'est le minimum requis pour se lancer dans ce petit projet! Cet article est largement "inspiré" d'un article anglais paru sur le site hackernoon.com ecrit par SANJAY GURUPRASAD.

Que va t-on faire ici?

Vous allez pouvoir creer votre premiere page web et heberger cette derniere en moins de 15 minute sans utiliser aucun un logiciel de programmation, on va utiliser uniquement du pur langage html/css. A la fin de ce petit tutoriel votre page web affichera une phrase selon votre choix sur un fond de couleur. Nous ne rentrerons volontairement pas dans un descriptif technique poussé afin de se concentrer sur la realisation du tutoriel.

Rendez vous sur GITHUB.COM

Pour commencer il faudra se rendre sur GITHUB.COM et creer un compte. Une fois le compte crée, cliquez sur "new repositories" pour faire un nouveau projet, il faudra ajouter un titre à votre projet (nous avons mis "ma premiere page"), n'oubliez pas de bien cocher la case " ADD a README file" et de valider le projet en cliquant sur "create repository".


Une fois toute ces etapes effectuées vous allez pouvoir passer à la prochaine tache pour faire vivre votre future oeuvre.


Creer un fichier index.html

C'est l'heure de "coder"! Vous aller creer un fichier que vous allez nommer index.html en cliquant sur le bouton "add file".


A l'interieur de ce fichier vide commencez par saisir ou copier coller le texte en rose qui se trouve apres cette capture d'ecran dans le fichier index.html . Il s'agit d'un texte html,ce code sert de base à toute conception de page web, c'est le squelette ! Ce texte html va afficher un texte et un titre via les balises "title" et "h1": MA PREMIERE PAGE. Mais vous pouvez bien evidemment afficher autre chose comme votre prenom ou une citation. 



        
            <!doctype html>
            <head>
             <meta charset="utf-8">  
              <title>MA PREMIERE PAGE</title>
            </head>
            <body>
                <h1>MA PREMIERE PAGE</h1> 
            </body>
            </html>

Une fois tout cela effectué vous pouvez appuyer sur le bouton "commit new file" pour soumettre votre fichier!


On passe sur github pages maintenant!

On attaque maintenant la partie "hebergement" il faut que le monde entier admire votre travail! Faites savoir à Github que vous souhaitez mettre en ligne votre creation sur "GITHUB PAGES".


Cliquez sur "settings" pour atteindre la partie " GITHUB PAGES SECTIONS" Selectionnez la partie "master branch", ensuite "root" en tant que source et sauvegardez en cliquant sur "save".


Le travail a été fait! Le lien de votre page apparait! Vous pouvez pouvez donc aller verifier ce que cela donne sur votre écran.


Un petit coup de CSS

Votre page a été publié mais il faut avouer que tout cela est tout de meme un peu triste, c'est l'heure de customiser votre boulot, c'est à partir d'ici que la partie CSS intervient. On utilise le css sur les pages web afin d'ajouter de la couleur, des effets, etc, c'est le maquillage du design web.



Il va falloir retourner sur votre fichier et le modifier en y ajoutant du css, pour cela vous devez vous rendre sur le fichier index.html et cliquer sur l'icone "edit".


Maintenant vous allez ajouter quelques lignes de CSS, pour ajouter un petit peu plus d'effet à votre oeuvre, pour cela copier coller ou saissisez le bloc de texte apres la capture d'ecran ci contre. Une fois que tout cela est fait N'oubliez pas de sauvegarder votre mise à jour en cliquant sur le bouton "Commit changes".



Le code CSS ci contre doit etre copié collé entre les deux balises suivantes:<meta charset="utf-8"> et 
 <title>MA PREMIERE PAGE </title>  
                   
                   <style>
            body{
        background-color: black;
        text-align: center;
        padding: 200px
        } 
      
       h1{
         font-size: 60px;
         color: #eeee33; 
         }
        </style>

Attendez quelques secondes afin que Github pages puisse prendre en compte le changement et ensuite.... Bim! Ca y'est! En moins de 10min vous avez pu sortir votre premiere page ! Satisfait?


C'est quoi la suite maintenant???

A vous de jouer en modifiant cette page comme vous le souhaitez, ajoutez des images, rajoutez du texte ou des animations etc... Pour approfondir le sujet et explorer l'univers du developpement informatique je vous invite à faire un tour sur les sites suivants: Udemy, Openclassroom ou freeCodeCamp.
